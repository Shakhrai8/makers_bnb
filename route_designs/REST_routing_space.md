# REST Routing (REST)
# Spaces
## Create a new space
Requests:
GET, POST /:id/:space_id
Response (200 0K):
It will create a new space
## Update the space
Requests:
PATCH /:id/:space_id/update
Response (200 0K):
It will update the space
## Delete the space
Requests:
DELETE /:id/:space_id/delete
Response (200 0K):
It will delete the space
